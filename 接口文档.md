### **用户管理接口文档**
#### 发送邮件
- **接口路径**: `/user/sendEmail`
- **请求方法**: `POST`
- **功能**: 添加新用户。
- **请求体**:
  queryString
```
String email
```
- 成功
    ```json
    {
      "code": 200,
      "message": "操作成功",
      "data": null
    }
    ```
  



####  注册用户
- **接口路径**: `/user/register`
- **请求方法**: `POST`
- **功能**: 添加新用户。
- **请求体**:
  application/x-www-form-urlencoded
  ```
  username  String
  password String
  code String 验证码
  ```

- **响应成功**:
    ```json
    {
      "code": 200,
      "message": "操作成功",
      "data": null
    }
    ```
  

#### 用户登录
- 路径："/user/login"
- 方法：POST
- 功能：如题
- 请求体：application/x-www-form-urlencoded
```
String username,
String password
```
- 响应数据:
```
    {
      "code": 200,
      "message": "操作成功",
      "data": token: String
    }
```
